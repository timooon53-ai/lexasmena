# lexasmena

Бот для работы с заказами такси и сопровождения поездок.

## Комиссия

Начисление считается как 15% от стоимости поездки (без привязки к числу успешных запросов).

## Доступ к боту

Если нет доступа, можно арендовать:

- На час — 1000₽
- На сутки — 5000₽
- Навсегда — 15000₽

После оплаты предоставляется полный доступ к заказу, аккаунты для заказа и сопровождение всей поездки.
Оплата возможна через оператора — @TakeMaxist.
